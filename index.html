Главная

Этот проект представляет собой систему для управления рыболовной флотилией. Система позволяет отслеживать информацию о траулерах, членах экипажа и результатах рыболовных рейсов. Основная цель - обеспечить эффективный сбор и анализ данных о деятельности флотилии.

Глоссарий

Траулер
Рыболовное судно, предназначенное для лова рыбы с помощью трала.
Банка
Мелководный участок моря, где производится лов рыбы.
Рейс
Поход судна в море для выполнения определенной задачи (в данном случае - лова рыбы).

Функциональные требования

Система должна предоставлять следующую информацию:

  • Перечень выполненных рейсов для указанного траулера за определенный период с информацией об общем количестве выловленной рыбы.
  • Перечень и количество выловленной рыбы по видам для указанной банки.
  • Сведения о датах рейсов и траулерах, посещавших банку, где было выловлено максимальное количество рыбы низкого качества.
  • Сведения о капитане и посещенных банках для траулера, выловившего наибольшее количество рыбы.
  • Перечень членов команды, которые должны быть отправлены на пенсию на указанную дату.

Спецификация проекта

Проект разработан на языке C++ и использует базу данных SQLite для хранения информации. Структура проекта включает следующие компоненты:

  • src: Исходный код приложения.
  • include: Заголовочные файлы.
  • database: SQL-скрипты для создания и наполнения базы данных.
  • tests: Модульные тесты.

Схема базы данных

База данных состоит из следующих таблиц:

  • trawlers: Информация о траулерах (название, водоизмещение, дата постройки).
  • crew_members: Информация о членах команды (фамилия, должность, дата приема на работу, год рождения).
  • trip_results: Результаты рейсов (название траулера, дата выхода и возвращения, посещенные банки, улов по видам рыб).

Подробная схема и скрипты находятся в папке database.
